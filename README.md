# Practice Use API

## Overview
This is just a practice to fetch, format, and render data from api. Also a practice to test codes.

## Setup
```shell
$ yarn build | yarn start  # to run the app in dev mode on localhost:3000

$ yarn test  # to launch the test runner
```

## Technologies
- TypeScript 4.4.4
- React 17.0.2
- Material UI 4.12.3
- Jest 26.6.0
- Axios 0.23.0
